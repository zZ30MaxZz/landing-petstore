1. **Estructura y contenido:**

   - Sección "Preguntas frecuentes (FAQ)" tipo acordeón con al menos 5 preguntas comunes.
   - Sección "Contacto" con formulario (nombre, email, teléfono, motivo de consulta, mensaje) y botón "Enviar". Incluir validación básica de campos.
   - Footer con información de contacto (dirección, teléfono, email), enlaces a redes sociales (íconos), y breve aviso legal.

   - Paleta de colores profesional: verdes suaves (#2E8B57, #98FB98), blancos y grises claros.
   - Tipografía legible y moderna (ej. 'Inter' o 'Roboto' desde Google Fonts).
   - Diseño móvil‑first, usando Flexbox y/o CSS Grid para layouts.
   - Efectos sutiles de hover y transiciones suaves en botones, enlaces y tarjetas.
   - Imágenes placeholders optimizadas (uso de `loading="lazy"` y atributos `alt` descriptivos).
   - Iconografía simple (puedes usar SVG embebido o una librería ligera como Feather Icons).

3. **Tecnologías y buenas prácticas:**
   - HTML5 semántico (header, nav, section, article, aside, footer).
   - CSS3 con variables (custom properties) para colores y espaciado.
   - JavaScript vanilla (sin frameworks) para funcionalidades interactivas (menú hamburguesa móvil, carrusel de testimonios, acordeón FAQ, validación de formulario).
   - Accesibilidad: roles ARIA, etiquetas label asociadas a inputs, contraste suficiente, navegación por teclado.
   - SEO básico: meta etiquetas title, description, viewport, y encabezados jerárquicos (H1‑H4).
   - Código limpio, comentado y organizado (separar CSS y JS en bloques claramente marcados si se entrega en un único archivo HTML, o proporcionar estructura de carpetas si se prefiere múltiples archivos).
